# NEXSTEP

Mobile-first PWA für geführte IT-Fehlerdiagnosen. NEXSTEP führt Nutzer durch kurze, verzweigte Prüfpfade, stellt Befehle kopierbar bereit und speichert abgeschlossene Fälle lokal auf dem Gerät.

## Entwicklung

```bash
npm install
npm run dev
```

## Qualität

```bash
npm run typecheck
npm test
npm run build
```

Die App wird statisch exportiert und über GitHub Pages veröffentlicht.
